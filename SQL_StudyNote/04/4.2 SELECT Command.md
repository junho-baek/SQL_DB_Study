# SELECT 문

---

## SELECT 문이란?

SQLite SELECT 문은 데이터베이스에서 데이터를 조회하는 가장 기본적인 명령어입니다.

---

## SELECT 문의 특징

- 결과로 행과 열이 있는 테이블을 반환
- 데이터베이스를 쿼리하는 데 사용
- 다양한 절(WHERE, ORDER BY 등)과 함께 사용 가능

---

## 기본 문법

기본적인 SELECT 문의 구조는 다음과 같습니다

```sql
SELECT 열 FROM 테이블명;
```

---

sqlite_sequence 는 오토업데이트 되는 테이블의 시퀀스 값을 저장하는 테이블입니다.
