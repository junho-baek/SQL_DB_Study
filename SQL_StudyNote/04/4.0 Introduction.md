# DML(Data Manipulation Language)이란?

데이터베이스의 데이터를 조작하는 언어로, SQL에서 가장 중요하고 자주 사용되는 명령어들의 집합입니다.

---

## DML의 주요 기능

1. 데이터 조회 (Query)

   - SELECT 문을 사용
   - 데이터 검색, 필터링, 정렬, 집계 등 수행
   - 가장 빈번하게 사용되는 명령어

2. 데이터 수정 (Update)
   - INSERT: 새로운 데이터 추가
   - UPDATE: 기존 데이터 수정
   - DELETE: 데이터 삭제

---

## DML의 중요성

- 실제 데이터 관리의 핵심 도구
- 데이터 분석과 리포팅의 기반
- 비즈니스 로직 구현의 필수 요소
