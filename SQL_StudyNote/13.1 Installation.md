# PostgreSQL 설치 방법

## 1. 기본 설치

- PostgreSQL 공식 홈페이지 방문
- macOS용 인스톨러 다운로드
- 인스톨러 실행하여 설치 완료

## 2. 함께 설치되는 도구들

- pgAdmin

  - MySQL Workbench와 유사한 GUI 관리 도구
  - 데이터베이스 시각적 관리 가능

- Stack Builder
  - PostgreSQL 확장 프로그램 설치 도구
  - 추가 기능 및 드라이버 설치 가능

## 3. 비밀번호 관리

- 비밀번호 분실 시
  - 공식 홈페이지 접속
  - 비밀번호 찾기 기능 이용
  - 절차에 따라 재설정 가능
