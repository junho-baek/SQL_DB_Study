# 13.0 Introduction

# PostgreSQL 소개

PostgreSQL은 세계에서 가장 진보된 오픈소스 관계형 데이터베이스 시스템입니다.

---

# 주요 특징

- 심플하면서도 복잡한 기능 제공
- 직관적인 데이터 타입 체계
- 다양한 내장 기능 보유
- 확장 가능한 익스텐션 시스템
- Python 코드 실행 지원

---

# MySQL과의 비교

- MySQL과 유사한 기본 기능 제공
- 더 많은 고급 기능 탑재
- 더 강력한 확장성

---

# 학습 범위

- PostgreSQL의 기본적인 사용법
- 범용적으로 활용 가능한 핵심 기능
- 실무에서 자주 사용되는 기능 위주

---

# 역사

- Ingres 데이터베이스를 계승
- Post-Ingres 프로젝트에서 발전
- 현대적인 RDBMS의 표준
